# 🍽️ Food-Lens Platform

Food-Lens is a **restaurant discovery and review platform** that helps users explore restaurants, read reviews, and discover dining options nearby.

The platform provides an **interactive search system, restaurant listings, ratings, and location mapping**, allowing users to easily find the best places to eat.

---

# 🚀 Features

## 🔍 Restaurant Search

Users can search restaurants quickly using the search bar.

Features:

* Keyword-based restaurant search
* Fast filtering
* Clean UI with search suggestions

---

## 🍜 Restaurant Listings

The homepage displays restaurants in a **card-based layout**.

Each restaurant card includes:

* Restaurant image
* Restaurant name
* Location
* Cuisine type
* Ratings
* Favorite & share options

---

## ⭐ Rating Filters

Users can filter restaurants based on rating levels.

Filters include:

* ⭐ 2+
* ⭐ 3+
* ⭐ 4+

This helps users quickly find **high-quality restaurants**.

---

## 📍 Restaurant Details Page

Each restaurant has a detailed page with complete information.

Information includes:

* Restaurant name
* Location
* Cuisine type
* Contact number
* Ratings and reviews
* Restaurant image gallery

---

## 🗺 Map Integration

The platform integrates **OpenStreetMap** to display the restaurant location.

Features:

* Interactive map
* Location pin for restaurant
* Easy navigation

---

## 🏙 Nearby Restaurants

Users can discover restaurants located near the selected restaurant.

Features:

* Nearby restaurant suggestions
* Similar cuisine recommendations

---

## ✍ Write Reviews

Users can write reviews for restaurants to share their dining experiences.
